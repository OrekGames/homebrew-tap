# OrekGames Homebrew Tap

Homebrew tap for installing OrekGames tools and libraries.

## Usage

```sh
brew tap OrekGames/tap
```

Then install any available formula:

```sh
brew install OrekGames/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `track` | CLI for issue tracking systems (YouTrack, Jira, GitHub, GitLab) |
